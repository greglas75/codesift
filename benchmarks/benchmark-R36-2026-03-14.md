## Critical Finding: Dedicated T9 Query Doesn't Help

Devoting all of Call B to T5+T7+T9 did NOT improve T9 coverage:
- R34 (T9 in Call B with T6-T10): 3 locations
- R35 (T9 in Call B with T5-T9): 3 locations
- **R36 (T9 with dedicated query): 4 locations** (+1 new, but overall same scale)

Each run surfaces different locations due to semantic randomness, but consistently finds 3-4.

**Conclusion**: T9 coverage is fundamentally capped at 4-6 locations with ANY Auggie query strategy. The remaining 14-16 locations require text/BM25 search.
