## Conversation Search

CodeSift indexes Claude Code conversation history for cross-referencing code with the discussions that shaped it.

### How it works
- Conversations stored as JSONL in `~/.claude/projects/{encoded-cwd}/` are parsed into searchable turn-pairs
- Auto-discovery at startup indexes current project conversations
- Session-end hook (`Stop` in `.claude/settings.local.json`) re-indexes after each session
- Noise filtered: tool_result dumps stripped, tool_use truncated, images replaced with `[image]`
- Compaction-aware: `isCompactSummary` skipped, last summary indexed as `conversation_summary`

### 3 New MCP Tools

| Tool | Purpose |
|------|---------|
| `index_conversations(project_path?)` | Index JSONL conversation files. Auto-detects current project if path omitted. |
| `search_conversations(query, project?, limit?)` | BM25F search over conversation turns. Returns session_id, timestamp, user_question, assistant_answer. |
| `find_conversations_for_symbol(symbol_name, repo)` | Cross-reference: find conversations that discussed a code symbol. |
